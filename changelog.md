# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-03
### Public Release
- Initial fully functional public release.
- Automated download of Twitch VODs and Chat using yt-dlp and TwitchDownloaderCLI.
- Automatic dependency management (ffmpeg, mpv, .NET Runtime, etc.).
- Chat rendering feature with transparent overlay support.
- Portable player generation (.bat) linking video and chat sync.
- Full offline support with portable .NET runtime integration.
- Persistent user configuration (OAuth tokens, channel preferences).
- Robust error handling for missing channels or empty VOD lists.
